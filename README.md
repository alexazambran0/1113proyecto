## 📚 Glosario del Proyecto
*   **Repositorio:** Espacio donde se almacena el código y la historia del proyecto.
*   **Rama (Branch):** Versión paralela del código para trabajar sin afectar la principal.
*   **Pull Request:** Propuesta para incluir cambios de una rama a otra.
*   **Merge:** Acción de combinar dos ramas de código.
